## 示例

使用破折号（`-`）或星号（`*`）作为项目符号。

- 项目符号列表项 1
- 项目符号列表项 2
- 项目符号列表项 3

1.  编号列表项 1。句点和第一个字母之间的两个空格
    有助于对齐。

2.  编号列表项 2。让我们在其中添加一个注释。

    > [!NOTE]: 我们做到了！

3.  编号列表项 3，其中包含代码块。在代码块之前
    需要一个空行。

    ```bash
    $ docker run hello-world
    ```

4.  编号列表项 4，其中包含项目符号列表和
    一个编号列表。

    - 子项 1
    - 子项 2

      1.  子子项 1
      2.  子子项 2，其中包含一个表格，因为我们喜欢热闹！
          缩进非常重要。

          | 表头 1 | 表头 2 |
          | -------- | -------- |
          | 内容 1  | 内容 2  |
          | 内容 3  | 内容 4  |

## 示例

### 基础表格

| 权限级别                                                         | 访问权限                                                     |
| :----------------------------------------------------------------------- | :------------------------------------------------------------ |
| 表格单元格中的**粗体**或_斜体_。下一个单元格故意留空。 |                                                               |
|                                                                          | 前一个单元格为空。等宽字体中的 `--flag`。                      |
| 读取                                                                     | 拉取                                                          |
| 读写                                                                | 拉取、推送                                                    |
| 管理员                                                                    | 以上所有权限，另加更新描述、创建和删除                        |

### 功能支持表格

| :--------- | :------------: |
| Ubuntu     |       ✅       |
| Debian     |       ✅       |
| Fedora     |                |
| Arch (btw) |       ✅       |
